## 🔯 伤寒钤法推演

《伤寒钤法》是一种根据患者**生年干支**（生肖属相）与**发病日干支**推算六经归经、匹配《伤寒论》条文及对应经方的古法，又称「算病法」。其核心是将五运六气与六经辨证融合，以运气推演直达经方。

**本系统的优化设计：**

* 只需输入**阳历日期**（年月日），自动换算天干地支，无需了解干支历法

* 实时显示对应年柱 / 日柱干支供参考

* 三重推演结果同时呈现：

  1. **属相法归经**（主推演）— 生年地支 × 发病日阴阳干
  2. **发病日六气归经**（辅助）— 六气歌推算
  3. **五运脏腑推演** — 天干化运，判断受病脏腑

* 支持填写发病时辰（12 时辰），进一步推演时柱六气

## 🌿 五运六气

基于《黄帝内经·素问》运气学说，提供三项查询功能：

* **年运查询** — 输入任意年份，推算五运六气、司天在泉、六步气位、二十四节气，附《内经》原文经文

* **个人体质** — 输入公历出生日期，推算先天运气体质、脏腑易损倾向、养生调护建议

* **天气预测** — 据五运太过/不及与司天在泉，推演当年气候特征与灾害倾向

## 🚀 部署到 GitHub Pages

### 方法一：直接上传（推荐）

1. 新建 GitHub 仓库（或使用现有仓库）
2. 将 `index.html` 上传到仓库根目录
3. 进入仓库 **Settings → Pages → Source**，选择 `main` 分支，点击 Save
4. 等待约 1 分钟后访问：

```
https://<你的用户名>.github.io/<仓库名>/
```

### 方法二：命令行部署

```bash
git clone https://github.com/<你的用户名>/<仓库名>.git
cd <仓库名>
cp shanghan_v3.html index.html
git add .
git commit -m "deploy: 伤寒论运气经方系统"
git push
```
